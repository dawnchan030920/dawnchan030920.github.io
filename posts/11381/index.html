软件清单

前言

我的电脑是我为数不多的安全感的来源，电脑配置的一大部分又是软件；为了延续我的安全感，也为了以后换新配置方便，把我目前能想到的——并不调查，以证实其亲密常用——陈列于此。

清单

办公

  1. Office 2019。希望以后可以用上Office 365。总之就是：配色的神！
  2. XMind ZEN。希望以后可以用上订阅版。思维导图也算是它一家独大的，轻量但功能不减。
  3. Edge(Chromium)。微软系浏览器，软粉只要爱就是了。就是好用。
  4. Typora。轻量级文档（Markdown）撰写，优雅而强大。
  5. PotPlayer。强大简洁；但是新版也开始有广告了。哎……
  6. DrawboardPDF。一个感觉不错的PDF阅读和编辑器，主要适配Ink Pen、触屏等现代玩法。一般Edge也够了。

开发

  1. Visual Studio 2019（含Blender）。不多说，就是强大。
  2. Visual Studio Code。不多说，就是好。
  3. Windows Terminal。从头到脚都是Windows气息的命令行工具。复古与现代的巧妙结合。
  4. StarUML。我是建模狂魔——因为图形好看。照旧，希望能买来。
  5. Figma。线框图不是很减压嘛？视觉美也是软件的一大组成部分。希望能用上Professional版。

环境

  1. Git。否则？
  2. .Net X。就是希望WinUI3和MAUI早点出。急死了！
  3. Node。没个Node还能搞事情？
  4. PowerShell。我觉得好；虽然有人说CmdLet破坏了命令行的简洁性，但我觉得反正有Alias；工程化的可拓展机制才是基石。

生活

  1. 网易云音乐。
  2. QQ和微信。

命令行工具

自己都不知道自己装了多少CLI工具。凭印象——上面也说了。

  1. GitHub CLI。没怎么用，但是觉得挺酷的。
  2. ScriptC#。不得先试过才知道？不过C# 9.0已经有可以替代的选项了。
  3. Chocolate之类的包安装工具。没怎么用，应该是一个简化流程的好东西。
  4. NPM（换源）。不然？
